Project One: PennMCITHub


A web platform specifically designed for students in the University of Pennsylvania's MCIT program. It enables collaboration through study buddy matching, GitHub activity tracking, and event scheduling. PennMCITHub fosters a connected learning environment tailored to the needs of tech-savvy students.


Tech Stack:

  • Frontend: React, Bootstrap
  • Backend: Node.js, Express
  • Database: MongoDB
  • Deployment: AWS (EC2 and S3)

Features:

  • Study Buddy Finder to match students based on shared courses and preferences.
  • GitHub activity tracking with integrated stats.
  • Event Scheduler for organizing workshops and group sessions.

Project Two: PathFinder


A platform to help senior high school students explore potential college majors based on their interests, skills, and career aspirations. It uses quizzes and data analytics to provide personalized recommendations.


Tech Stack:

  • Frontend: Angular, Material UI
  • Backend: Flask, Python
  • Database: PostgreSQL
  • Data Visualization: Chart.js
  • Deployment: Heroku

Features:

  • Interactive quizzes to assess interests and skills.
  • Career insights including salary trends and required skills.
  • College comparison tool for tuition, location, and acceptance rates.

Project Three: PlanMyGapYear


A gap year planning tool that helps high school and college graduates explore opportunities like internships, volunteering, or travel programs before college.


Tech stack:

  • Frontend: Vue.js, TailwindCSS
  • Backend: Django
  • Database: SQLite
  • APIs: Google Maps API
  • Deployment: Netlify

Features:

  • Opportunity database for internships, volunteer programs, and travel options.
  • Budget calculator to estimate gap year costs.
  • Interactive map with filters for region and type of opportunity.
